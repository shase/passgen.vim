# passgen.vim

This plugin to generate passwords. 

## Requirements
Python3

## Install
Example for `vim-plug`
```
Plug 'shase/passgen.vim'
```

Example for `dein`
```
call dein#add('shase/passgen.vim')
```

## Usage
Generate a password for the number of characters in the argument and insert buffer
```
:PASSGEN 10
```
